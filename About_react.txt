						
	■ 리액트					
						
		리액트는 페이스북이 만든 사용자 UI 구축을 위한 라이브러리입니다. 오직 사용자의 View에만 초점을 맞추고 있죠.				
		그래서 Routing 같은 기술이 리액트 자체에는 들어가있지 않습니다만, 개발자들이 여러가지 리액트에 필요한 것들을 만들어 놓아서,				
		사실상 프레임워크 위치에 있다고 봐도 무방할 정도입니다.				
						
		※ 특징				
			1. JSX 문법			
				JSX는 JavaScript 안에서 HTML 문법을 사용해서 view를 구성할 수 있게 도와주는 JavaScript 문법입니다.		
					** 컴포넌트에 여러 태그가 있다면 반드시 부모 태그로 감싸주어야합니다.	
				컴포넌트 작성 후 앱을 실행할 때, 리액트는 JSX 문법을 순수 자바스크립트로 컴파일을 합니다.		
			2. Component 기반			
				리액트는 컴포넌트 기반 라이브러리 인데요, 컴포넌트 기반이라 함은 기존의 웹 페이지를 작성할 때처럼 하나의 HTML 코드를 집어넣고 하는 것이 아닌,		
				여러 부분을 분할해서 코드의 재사용성과 유지보수성을 증가시켜줍니다.		
				쉽게 설명해보자면,		
					우리가 프론트 부분을 짤 때 아주 긴 HTML 코드를 적을 때가 있습니다. 그 코드의 일부를 수정한다고 할때, 해당 부분을 찾는 것부터 시작해서 	
					내가 수정한 코드의 부분만 수정이 되지 않고 다른 부분을 건드리게 되는 등의 문제가 생길 수 있습니다.	
				컴포넌트 기반인 리액트로 개발을 하면 HTML 코드를 부분적으로 파일로 담게 되고, 필요한 부분의 파일만 수정하면 됩니다.		
				리액트 컴포넌트는 함수형과 클래스형 2가지 종류로 구성되어 있습니다.		
			3. Virtual DOM			
				virtual DOM은 기존 DOM의 한계를 탈피하기 위해서 나온 대안입니다.		
					** DOM (Document Object Model) : 문서객체모델, 우리가 사용하는 HTML 단위 하나하나를 객체로 생각한 모델이라고 보면 됩니다.	
						
		※ 폴더 및 파일 설명				
			public 폴더			
				index.html 		
					리액트에서는 html 페이지 하나로 웹 앱을 만듭니다. (Single page application, 단일페이지)	
					그리고 앞으로 작성할 모든 리액트 컴포넌트를 이 html 파일에서 렌더(Render) 하며, 기본적으로 이 index.html 파일을 html 템플릿이라고 합니다.	
					index.html 파일에는 id가 "root" 인 div 태그가 있는데, 모든 리액트 컴포넌트들은 이 div 태그 안에서 렌더됩니다.	
				index.js		
					리액트 앱에 존재하는 render 중 최상위에 있는 render가 존재하는 파일입니다. 	
					이 파일에는 React.DOM.render() 가 있는데, 	
						이는 App.js 파일에서 렌더한 App 컴포넌트를 index.html 파일에 있는 id가 "root" 인 div 태그 안에서 실행하라는 뜻입니다.
			src 폴더			
				이 폴더에는 리액트 앱 전체(global, 전역)에 적용되는 js파일과 css파일이 존재합니다. 앞으로 작성할 모든 리액트 컴포넌트들은 이 폴더 안에 생성할 것입니다.		
				App.js		
					루트(root) 컴포넌트라고 불리며, 이 파일에는 브라우저에서 실제로 렌더되는 컴포넌트들을 포함하고 있습니다.	
					App.js 파일은 JSX라는 확장자 파일을 return 하는데, JSX는 JavaScript를 확장한 문법입니다.	
					$' 표시가 없는 jsp 파일과 유사하다고 생각하면 될 것 같습니다.	
					다만, 리액트가 JavaScript와 JSX 문법을 헷갈리지 않게 하기 위해 속성값은 다를 수 있습니다.	
						(예, js의 class를 사용하지 않고, className을 사용합니다.)
						
		※ 각 컴포넌트들은 props와 state를 통해 데이터를 입력받아 브라우저에서 이를 렌더링하여 UI를 통해 결과물을 출력해줍니다.
